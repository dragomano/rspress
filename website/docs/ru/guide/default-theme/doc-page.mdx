# Страница документации

## Тип страницы

Вы можете указать `pageType` в блоке метаданных, чтобы использовать разные макеты страницы. По умолчанию тип страницы документации — `doc`, поэтому слева отображается боковое меню (`sidebar`), а справа — оглавление (`outline`). Если они вам не нужны, установите `pageType: custom`:

```yaml
---
pageType: custom
---
```

## Сайдбар

В файле `rspress.config.ts` можно настроить содержимое сайдбара. Подробности — в разделе [конфигурация `sidebar`](/api/config/config-theme#sidebar).
