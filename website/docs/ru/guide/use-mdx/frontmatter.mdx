# Метаданные

Вы можете добавить метаданные в начало вашего Markdown-файла. Это особая часть, которая использует формат [YAML](https://yaml.org/) для определения метаданных и обрамляется тремя дефисами `---` сверху и снизу.

Например, с помощью [title](/api/config/config-frontmatter#title) можно указать заголовок страницы:

```yaml
---
title: Привет, мир
---
```

Вы также можете обращаться к свойствам, объявленным в блоке метаданных, прямо в теле документа, например:

```markdown
---
title: Привет, мир
---

# {frontmatter.title}
```

Ранее объявленные свойства будут переданы в компонент как свойства объекта `frontmatter`. Таким образом, итоговый вывод будет выглядеть так:

```html
<h1>Привет, мир</h1>
```

:::tip Совет
Список доступных метаданных смотрите в разделе [Конфигурация метаданных](/api/config/config-frontmatter).
:::
