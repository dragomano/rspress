# Контейнеры

Контейнеры — отличный способ выделить важную информацию и дать подсказки пользователям.

:::tip Подсказка
Rspress поддерживает два стиля синтаксиса: [синтаксис с тремя двоеточиями `:::`](#three-colon-syntax) и [синтаксис GitHub Markdown Alerts](#github-markdown-alerts-syntax).
:::

## Синтаксис `:::` \{#three-colon-syntax}

С помощью синтаксиса `:::` можно создавать пользовательские контейнеры и задавать собственные заголовки. Например:

import { Tabs, Tab } from '@theme';

<Tabs>

<Tab label="Отрендеренный результат">

:::note
Это `блок` типа `note`
:::

:::tip
Это `блок` типа `tip`
:::

:::info
Это `блок` типа `info`
:::

:::warning
Это `блок` типа `warning`
:::

:::danger
Это `блок` типа `danger`
:::

:::details
Это `блок` типа `details`
:::

:::tip Пользовательский заголовок
Это `блок` с `пользовательским заголовком`
:::

:::tip\{title="Пользовательский заголовок"}
Это `блок` с `пользовательским заголовком`
:::

</Tab>

<Tab label="Синтаксис">

```markdown
:::note
Это `блок` типа `note`
:::

:::tip
Это `блок` типа `tip`
:::

:::info
Это `блок` типа `info`
:::

:::warning
Это `блок` типа `warning`
:::

:::danger
Это `блок` типа `danger`
:::

:::details
Это `блок` типа `details`
:::

:::tip Пользовательский заголовок
Это `блок` с `пользовательским заголовком`
:::

:::tip{title="Пользовательский заголовок"}
Это `блок` с `пользовательским заголовком`
:::
```

</Tab>

</Tabs>

:::warning Примечание

- Типы контейнеров должны быть в **нижнем регистре**. Используйте `:::tip`, `:::warning`, `:::caution` и т. д. Типы с заглавной буквы (`:::Tip`, `:::Warning`) распознаваться не будут.

- При использовании синтаксиса `:::` в файлах `.mdx` и задании собственного заголовка через фигурные скобки их нужно экранировать. Поэтому рекомендуется сразу писать `:::tip Название заголовка`.

```mdx
:::tip\{title="Пользовательский заголовок"}
Это `блок` с `пользовательским заголовком`
:::
```

:::

## Синтаксис блоков уведомлений GitHub Markdown \{#github-markdown-alerts-syntax}

Вы можете использовать [синтаксис блоков уведомлений GitHub Markdown](https://docs.github.com/ru/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax#alerts) для создания контейнеров.

<Tabs>

<Tab label="Отрендеренный результат">

> [!NOTE]
> Это `блок` типа `note`

> [!TIP]
> Это `блок` типа `tip`

> [!INFO]
> Это `блок` типа `info`

> [!WARNING]
> Это `блок` типа `warning`

> [!DANGER]
> Это `блок` типа `danger`

> [!DETAILS]
> Это `блок` типа `details`

</Tab>

<Tab label="Синтаксис">

```markdown
> [!NOTE]
> Это `блок` типа `note`

> [!TIP]
> Это `блок` типа `tip`

> [!INFO]
> Это `блок` типа `info`

> [!WARNING]
> Это `блок` типа `warning`

> [!DANGER]
> Это `блок` типа `danger`

> [!DETAILS]
> Это `блок` типа `details`
```

</Tab>

</Tabs>
