# Обзор

## Официальные плагины

Официальные плагины включают:

- [@rspress/plugin-llms](./llms): Генерирует файлы по спецификации [llms.txt](https://llmstxt.org/) для вашего сайта на Rspress — помогает большим языковым моделям лучше понимать содержимое документации.
- [@rspress/plugin-medium-zoom](./medium-zoom): Интеграция [medium-zoom](https://github.com/francoischalifour/medium-zoom) — позволяет увеличивать изображения по клику.
- [@rspress/plugin-client-redirects](./client-redirects): Поддержка клиентских редиректов.
- [@rspress/plugin-last-updated](./last-updated): Отображение времени последнего обновления статьи.
- [@rspress/plugin-typedoc](./typedoc): Интеграция с [TypeDoc](https://github.com/TypeStrong/typedoc) — автоматическая генерация API-документации для TypeScript-модулей.
- [@rspress/plugin-api-docgen](./api-docgen): Интеграция с [react-docgen-typescript](https://github.com/styleguidist/react-docgen-typescript) и [documentation](https://github.com/documentationjs/documentation) — автоматическое создание описаний API для React-компонентов и JS/TS-кода.
- [@rspress/plugin-preview](./preview): Поддержка предварительного просмотра блоков кода в Markdown/MDX.
- [@rspress/plugin-playground](./playground): Интерактивная «песочница» (playground) с живым предпросмотром кода из Markdown/MDX-файлов.
- [@rspress/plugin-rss](./rss): Генерация RSS-лент с помощью библиотеки [feed](https://github.com/jpmonette/feed) для сайта документации.
